# ETL E-commerce Data Pipeline

## 📌 Sobre o Projeto

Projeto de ETL desenvolvido utilizando Python, Pandas, PostgreSQL e Power BI com base no dataset público de e-commerce brasileiro da Olist.

O objetivo do projeto foi realizar:
- extração de dados CSV
- tratamento e transformação de dados
- carga em banco PostgreSQL
- consultas SQL
- visualização analítica no Power BI

---

# 🚀 Tecnologias Utilizadas

- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- psycopg2
- Power BI

---

# 📂 Pipeline do Projeto

CSV Dataset
    ↓
Python + Pandas
    ↓
ETL / Data Cleaning
    ↓
PostgreSQL
    ↓
SQL Queries
    ↓
Power BI Dashboard
